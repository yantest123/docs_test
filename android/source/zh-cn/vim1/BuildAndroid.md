### 编译 Android

```bash
$ cd PATH_YOUR_PROJECT
$ source build/envsetup.sh
$ lunch TARGET_LUNCH
$ make -jN otapackage
```

{% note info 注意 %}

* 把N换成数字，如你电脑CPU的线程数。
* TARGET_LUNCH定义如下：
  * 编译Android6.0时: kvim-userdebug-32
  * 编译Android7.1时: kvim-userdebug-64
  * 编译Android9.0时: kvim-userdebug

生成镜像文件如下：

* out/target/product/kvim/update.img

{% endnote %}
